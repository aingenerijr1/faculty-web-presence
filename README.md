# Faculty Course Support Hub

## Project Purpose

The Faculty Course Support Hub is a two-page faculty web presence prototype for Professor Dawn Ellis. The site gives students one clear place to find instructor support information, office hours, course details, and a structured way to request help.

The design also demonstrates how a school-facing product could be customized for different institutions. School colors are controlled through CSS variables in the shared stylesheet, making the visual theme easy to adjust for future clients.

## Repository Contents

- `index.html` - Front-facing faculty homepage with profile information, courses, office hours, and support links.
- `contact.html` - Information-gathering student support form page.
- `css/styles.css` - Shared stylesheet with reusable layout rules and school color variables.
- `assets/faculty-hero.svg` - Supporting visual asset used on the homepage.
- `README.md` - Project purpose, file guide, and local viewing instructions.

## How to View the Pages Locally

1. Download or clone this repository to your computer.
2. Open the project folder.
3. Double-click `index.html` to open the homepage in a web browser.
4. Use the navigation links to move between the homepage and the student support form.

No JavaScript or backend setup is required. The form is included for layout and design demonstration only; it does not collect, submit, or store real data.

## Design Notes

The prototype follows the assignment wireframe direction with a front-facing faculty page and an information-gathering page. The visual style uses a professional school palette, responsive layouts, cards, a prominent office-hours section, and consistent navigation across both pages.

To customize the design for another school, edit the color variables at the top of `css/styles.css`.
